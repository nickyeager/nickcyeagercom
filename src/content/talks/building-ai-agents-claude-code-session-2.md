## What You'll Build

A communications and scheduling system for MeetupBot, driven by PRPs (Product Requirements Prompts).

## Key Patterns

- **Context engineering > prompt engineering** — complete blueprints beat clever phrasing
- **PRPs** — implementation specs with validation gates, not just requirements docs
- **Slash commands** — composable agent capabilities for generating and executing PRPs
